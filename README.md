Plant-Dex API

This is the backend for Plant-Dex. A live version of the app
can be found at https://plant-dex.vercel.app/

Back End

    -Node and Express
        -Authentication via JWT
        -RESTful Api

    -Testing
        -Supertest (integration)
        -Mocha and Chai (unit)

    -DataBase    
        -Postgres
        -Knex.js

Production

    -Deployed using Heroku
